# Contact Management System

A full-stack web application to manage user contacts with registration, login, profile management, and password change functionality.


# Features

- User registration with email or phone
- User login with email or phone
- View user profile
- Change password
- Manage contacts (add, view, delete)
- RESTful backend using Spring Boot
- Frontend using React.js

# Tech Stack

- Backend: Java, Spring Boot, Spring Data JPA, MySQL
- Frontend: React.js, CSS
- Database: MySQL
